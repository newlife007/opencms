## 1. 系统概述

### 1.1 什么是OpenWan？

OpenWan是一套专业的媒体资产管理系统（Media Asset Management），用于管理广播电视台、企业、教育机构的视频、音频、图片和富媒体资源。

### 1.2 主要功能

- 📁 **文件管理**: 上传、编目、分类、搜索、发布
- 👥 **用户管理**: 用户、用户组、角色、权限管理
- 🔍 **全文搜索**: 快速检索媒体资源
- 🎬 **视频转码**: 自动转码生成预览
- 📊 **工作流**: 审批、发布流程管理
- 🔒 **权限控制**: 基于角色的访问控制（RBAC）

### 1.3 系统要求

**浏览器支持**:
- Chrome 90+ (推荐)
- Firefox 88+
- Safari 14+
- Edge 90+

**网络要求**:
- 上传带宽: ≥10 Mbps (建议100 Mbps)
- 下载带宽: ≥5 Mbps

## 4. 功能模块

### 4.1 文件管理

#### 4.1.1 文件上传

**步骤**:
1. 点击左侧菜单"文件管理" → "文件上传"
2. 点击上传区域或拖拽文件到上传区
3. 选择文件分类
4. 填写文件描述（可选）
5. 选择浏览等级（谁可以查看）
6. 选择访问组（哪些组可以访问）
7. 点击"开始上传"

**支持的文件类型**:

| 类型 | 支持格式 | 最大大小 |
|-----|---------|---------|
| 视频 | MP4, AVI, MOV, FLV, MKV, WMV | 2GB |
| 音频 | MP3, WAV, AAC, FLAC, WMA | 500MB |
| 图片 | JPG, PNG, GIF, BMP, TIFF | 50MB |
| 文档 | PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX | 100MB |

**上传注意事项**:
- ✅ 文件名使用中英文、数字、下划线
- ✅ 避免特殊字符（如 `/ \ : * ? " < > |`）
- ✅ 大文件建议使用有线网络上传
- ⚠️ 上传过程中不要关闭浏览器

#### 4.1.2 文件浏览

**查看文件列表**:
1. 点击"文件管理" → "文件浏览"
2. 使用左侧过滤器筛选：
   - 文件类型（视频/音频/图片/文档）
   - 文件状态（新建/待审/已发布/已拒绝）
   - 分类目录
   - 上传日期范围
   - 上传人

**视图模式**:
- **网格视图**: 显示缩略图，适合浏览大量文件
- **列表视图**: 显示详细信息，适合精确查找

**排序方式**:
- 按上传时间（默认）
- 按文件名
- 按文件大小
- 按更新时间

#### 4.1.3 文件编目

文件上传后需要编目（填写元数据）才能更好地管理和检索。

**编目步骤**:
1. 在文件列表中找到待编目文件（状态：新建）
2. 点击"编辑"按钮
3. 填写元数据表单：
   - **基本信息**: 标题、描述、关键词
   - **分类信息**: 主分类、子分类、标签
   - **版权信息**: 作者、版权所有者、许可类型
   - **技术信息**: 拍摄日期、地点、设备
   - **自定义字段**: 根据分类动态显示

4. 点击"保存"或"保存并提交审核"

**元数据字段说明**:

| 字段名 | 必填 | 说明 | 示例 |
|-------|------|------|------|
| 标题 | ✅ | 文件标题 | "2026春节联欢晚会" |
| 描述 | ❌ | 详细描述 | "第44届春节联欢晚会完整版" |
| 分类 | ✅ | 文件分类 | "电视节目/综艺/晚会" |
| 关键词 | ❌ | 搜索关键词 | "春晚,综艺,娱乐" |
| 作者 | ❌ | 创作者 | "中央电视台" |
| 拍摄日期 | ❌ | 内容拍摄日期 | "2026-01-28" |
| 浏览等级 | ✅ | 访问权限 | "公开/内部/机密" |

#### 4.1.4 文件搜索

**快速搜索**:
1. 在页面顶部搜索框输入关键词
2. 按Enter或点击搜索图标
3. 系统自动搜索标题、描述、关键词

**高级搜索**:
1. 点击搜索框右侧"高级搜索"
2. 设置多个搜索条件：
   - 关键词
   - 文件类型
   - 分类
   - 上传日期范围
   - 文件大小范围
   - 上传人
   - 状态
3. 点击"搜索"

**搜索技巧**:
- 使用空格分隔多个关键词（AND逻辑）
- 使用引号精确匹配：`"春节联欢晚会"`
- 使用减号排除：`春晚 -重播`
- 通配符搜索：`春*晚` (匹配"春节联欢晚会"等)

#### 4.1.5 文件预览

**视频预览**:
1. 点击文件卡片或文件名
2. 进入文件详情页
3. 视频播放器自动加载预览文件
4. 播放器功能：
   - ▶️ 播放/暂停
   - ⏩ 快进/快退
   - 🔊 音量控制
   - ⛶ 全屏播放
   - ⚙️ 播放速度调节

**支持的预览格式**:
- 视频: FLV (H.264 + AAC)
- 音频: MP3, AAC
- 图片: JPG, PNG, GIF
- 文档: PDF (内嵌查看器)

#### 4.1.6 文件下载

**下载原文件**:
1. 进入文件详情页
2. 点击"下载"按钮
3. 浏览器开始下载

**权限说明**:
- ✅ 文件设置了"允许下载"
- ✅ 您的用户组有下载权限
- ✅ 您的浏览等级 ≥ 文件等级
- ❌ 否则显示"无权限下载"

#### 4.1.7 文件删除

**删除文件**:
1. 在文件列表中选择要删除的文件
2. 点击"删除"按钮
3. 确认删除操作

**注意事项**:
- ⚠️ 删除后文件进入回收站（状态：已删除）
- ⚠️ 回收站文件保留30天
- ⚠️ 30天后自动永久删除
- ⚠️ 管理员可以恢复或永久删除

### 4.2 工作流管理

#### 4.2.1 提交审核

内容编辑完成编目后，需要提交审核才能发布。

**提交步骤**:
1. 完成文件编目
2. 点击"提交审核"按钮
3. 填写审核说明（可选）
4. 点击"确定"
5. 文件状态变为"待审核"

#### 4.2.2 审核文件

审核人员审核待审文件。

**审核步骤**:
1. 点击"工作流" → "待审文件"
2. 查看待审文件列表
3. 点击文件查看详情
4. 审核操作：
   - **通过**: 文件变为"已发布"，对用户可见
   - **拒绝**: 文件变为"已拒绝"，退回编辑
   - **暂缓**: 文件保持"待审核"，留待进一步讨论

5. 填写审核意见（必填）
6. 点击"确定"

**审核意见模板**:
- 通过: "内容符合规范，同意发布"
- 拒绝: "标题不规范，请修改为XXX"
- 暂缓: "需要进一步核实版权信息"

#### 4.2.3 工作流统计

查看工作流统计数据：
- 待审文件数量
- 今日已审数量
- 本周已审数量
- 审核通过率
- 平均审核时长

### 4.3 分类管理

#### 4.3.1 分类树结构

OpenWan使用树形分类结构，支持多级分类。

**示例分类结构**:
```
媒体资源
├── 电视节目
│   ├── 新闻
│   │   ├── 新闻联播
│   │   ├── 焦点访谈
│   │   └── 地方新闻
│   ├── 综艺
│   │   ├── 晚会
│   │   ├── 访谈
│   │   └── 真人秀
│   └── 电视剧
│       ├── 古装剧
│       ├── 现代剧
│       └── 悬疑剧
├── 广告素材
│   ├── 企业宣传片
│   ├── 产品广告
│   └── 公益广告
└── 教育培训
    ├── 员工培训
    ├── 产品培训
    └── 技术培训
```

#### 4.3.2 创建分类

**权限**: 系统管理员、分类管理员

**创建步骤**:
1. 点击"系统管理" → "分类管理"
2. 点击"新增分类"或在现有分类上点击"添加子分类"
3. 填写分类信息：
   - 分类名称 (必填)
   - 上级分类 (选择父分类，留空则为根分类)
   - 分类描述
   - 排序权重 (数字越大越靠前)
   - 图标 (可选)
   - 状态 (启用/禁用)
4. 设置访问权限：
   - 浏览等级要求
   - 允许访问的用户组
5. 点击"保存"

#### 4.3.3 管理分类

**编辑分类**:
1. 在分类列表中找到要编辑的分类
2. 点击"编辑"按钮
3. 修改分类信息
4. 点击"保存"

**移动分类**:
- 拖拽分类到新的父分类下
- 或编辑分类，修改"上级分类"

**删除分类**:
1. 点击分类的"删除"按钮
2. 确认删除操作
3. ⚠️ 注意：
   - 如果分类下有子分类，需先删除子分类
   - 如果分类下有文件，需要重新分配或一并删除

### 4.4 用户管理

#### 4.4.1 用户列表

查看所有用户：
1. 点击"系统管理" → "用户管理"
2. 查看用户列表，包含：
   - 用户名
   - 邮箱
   - 所属用户组
   - 状态（启用/禁用）
   - 创建时间
   - 最后登录时间

#### 4.4.2 创建用户

**权限**: 系统管理员

**创建步骤**:
1. 点击"新增用户"按钮
2. 填写用户信息：
   - 用户名 (必填，3-20个字符，字母数字下划线)
   - 密码 (必填，至少8位，包含大小写字母和数字)
   - 确认密码
   - 邮箱 (必填，用于找回密码)
   - 真实姓名 (可选)
   - 手机号 (可选)
   - 所属用户组 (必填，下拉选择)
   - 浏览等级 (必填，决定可访问的文件范围)
   - 状态 (启用/禁用)
3. 点击"保存"

**用户名规则**:
- ✅ 3-20个字符
- ✅ 字母、数字、下划线
- ❌ 不能以数字开头
- ❌ 不能使用特殊字符

**密码规则**:
- ✅ 至少8位
- ✅ 包含大写字母
- ✅ 包含小写字母
- ✅ 包含数字
- ✅ 可包含特殊字符 `!@#$%^&*()`

#### 4.4.3 编辑用户

1. 在用户列表中点击"编辑"
2. 修改用户信息
3. 点击"保存"

**可修改字段**:
- 邮箱
- 真实姓名
- 手机号
- 所属用户组
- 浏览等级
- 状态

**不可修改字段**:
- 用户名 (创建后不可修改)

#### 4.4.4 重置密码

**管理员重置**:
1. 在用户列表中点击"重置密码"
2. 输入新密码（或系统生成）
3. 将新密码告知用户

**用户自助重置**:
1. 登录后进入"个人中心"
2. 点击"修改密码"
3. 输入旧密码和新密码
4. 点击"确定"

#### 4.4.5 禁用/启用用户

**禁用用户**:
1. 在用户列表中找到用户
2. 点击"禁用"按钮
3. 确认操作
4. 用户将无法登录

**启用用户**:
1. 找到已禁用的用户
2. 点击"启用"按钮
3. 用户可以正常登录

### 4.5 用户组管理

#### 4.5.1 用户组概念

用户组是用户的集合，用于批量管理权限。

**系统预置用户组**:
- **管理员组**: 拥有所有权限
- **编辑组**: 可以上传、编目、提交审核
- **审核组**: 可以审核文件
- **查看组**: 只能查看和下载

#### 4.5.2 创建用户组

1. 点击"系统管理" → "用户组管理"
2. 点击"新增用户组"
3. 填写信息：
   - 组名 (必填)
   - 描述
   - 状态
4. 点击"保存"

#### 4.5.3 分配分类权限

控制用户组可以访问哪些分类。

1. 在用户组列表中点击"分类权限"
2. 勾选允许访问的分类
3. 支持多选
4. 点击"保存"

#### 4.5.4 分配角色

控制用户组拥有哪些角色（功能权限）。

1. 在用户组列表中点击"角色分配"
2. 勾选要分配的角色：
   - 内容管理角色
   - 审核角色
   - 系统管理角色
3. 点击"保存"

#### 4.5.5 查看组成员

1. 在用户组列表中点击"成员"
2. 查看组内所有用户
3. 可以添加或移除成员

## 5. 常见问题

### 5.1 登录问题

**Q: 忘记密码怎么办？**

A: 
1. 点击登录页面"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件
4. 如果没有收到邮件，联系管理员重置

**Q: 账号被锁定怎么办？**

A: 
- 连续输错密码5次会被锁定30分钟
- 等待30分钟后自动解锁
- 或联系管理员手动解锁

**Q: 提示"该账号已被禁用"？**

A: 
- 联系管理员确认原因
- 管理员可在用户管理中启用账号

---

### 5.2 上传问题

**Q: 上传失败怎么办？**

A: 可能原因：
1. 文件太大（超过限制）
   - 解决：压缩文件或分段上传
2. 网络不稳定
   - 解决：使用有线网络，关闭其他下载任务
3. 文件格式不支持
   - 解决：转换为支持的格式
4. 磁盘空间不足
   - 解决：联系管理员清理空间

**Q: 上传速度很慢？**

A: 
- 检查网络带宽
- 使用有线网络代替Wi-Fi
- 避开网络高峰期（如上午10点-11点）
- 关闭其他占用带宽的程序

**Q: 上传的视频没有生成预览？**

A: 
- 等待几分钟，转码需要时间（根据文件大小）
- 查看文件详情页，查看转码状态
- 如果转码失败，联系管理员查看日志

---

### 5.3 搜索问题

**Q: 搜索不到刚上传的文件？**

A: 
- 搜索索引更新有延迟（约5分钟）
- 等待一会儿再试
- 或使用"文件浏览"按分类查找

**Q: 搜索结果不准确？**

A: 
- 使用"高级搜索"设置更多条件
- 使用引号精确匹配：`"完整标题"`
- 检查关键词拼写

**Q: 搜索没有结果？**

A: 
- 简化搜索词（去掉修饰词）
- 使用同义词
- 检查文件是否已发布（只有已发布文件才能被搜索）

---

### 5.4 权限问题

**Q: 提示"无权限访问"？**

A: 可能原因：
1. 您的浏览等级不够
2. 您的用户组没有该分类权限
3. 文件仅限特定组访问
4. 联系管理员授权

**Q: 无法下载文件？**

A: 
- 文件可能未开启下载权限
- 您的用户组可能无下载权限
- 联系管理员确认

**Q: 无法看到某个菜单？**

A: 
- 您的角色可能没有该功能权限
- 联系管理员分配权限

---

### 5.5 播放问题

**Q: 视频无法播放？**

A: 
1. 检查浏览器是否为最新版本
2. 尝试刷新页面（Ctrl+F5）
3. 清除浏览器缓存
4. 尝试其他浏览器（推荐Chrome）
5. 检查网络连接

**Q: 视频播放卡顿？**

A: 
- 检查网络速度
- 降低播放质量（如有多个清晰度）
- 暂停等待缓冲
- 使用有线网络

**Q: 音频无法播放或无声音？**

A: 
- 检查音量设置（浏览器和系统）
- 检查是否静音
- 尝试其他文件确认是否为个别文件问题

## 6. 最佳实践

### 6.1 文件命名规范

**推荐命名格式**:
```
[日期]_[分类]_[标题]_[版本].[扩展名]

示例:
20260207_新闻_春节特别报道_v1.mp4
20260207_综艺_春晚节目单_终版.pdf
```

**命名原则**:
- ✅ 使用有意义的名称
- ✅ 包含日期便于排序
- ✅ 标明版本避免混淆
- ❌ 避免纯数字文件名
- ❌ 避免特殊字符

---

### 6.2 元数据规范

**标题**:
- 简洁明确，50字以内
- 包含关键信息（时间、地点、事件）
- 例: "2026年中央电视台春节联欢晚会"

**描述**:
- 详细说明内容（200-500字）
- 包含5W1H（何时、何地、何人、何事、为何、如何）
- 段落清晰，便于阅读

**关键词**:
- 3-10个关键词
- 逗号分隔
- 包含主题词、地名、人名、机构名
- 例: "春晚,综艺,娱乐,中央电视台,2026"

---

### 6.3 分类管理

**分类层级**:
- 建议不超过4级
- 第1级：大类（如"电视节目"）
- 第2级：中类（如"新闻"）
- 第3级：小类（如"新闻联播"）
- 第4级：细类（如"2026年"）

**分类数量**:
- 每级不超过20个分类
- 太多会导致选择困难
- 可用标签补充分类

---

### 6.4 权限管理

**最小权限原则**:
- 只授予必需的权限
- 定期审查权限
- 及时回收离职人员权限

**用户组设计**:
- 按部门或职能划分
- 避免过度细分
- 定期整理合并

---

### 6.5 备份与归档

**重要文件备份**:
- 手动下载重要文件到本地
- 定期导出元数据
- 异地存储一份副本

**文件归档**:
- 定期整理旧文件
- 不常用文件移到归档分类
- 删除重复或无用文件

---

### 6.6 性能优化

**上传优化**:
- 使用有线网络
- 避开网络高峰期
- 大文件建议分段上传
- 上传前压缩文件

**搜索优化**:
- 使用精确关键词
- 利用高级搜索过滤
- 按分类缩小范围
- 保存常用搜索
