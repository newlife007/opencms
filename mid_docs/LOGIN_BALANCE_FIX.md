## 📐 布局说明

### Flexbox布局

```css
.login-container {
  display: flex;  /* 启用flexbox */
}

.login-left {
  flex: 1;  /* 占据1份空间 (50%) */
  min-width: 0;  /* 允许收缩 */
}

.login-right {
  flex: 1;  /* 占据1份空间 (50%) */
  min-width: 0;  /* 允许收缩 */
}
```

**说明**：
- 两个区域的 `flex: 1` 相同
- 因此平均分配容器宽度
- 各占50%空间
