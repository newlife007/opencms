## 🔧 技术实现

### CSS Flexbox比例

```css
/* 左侧：flex: 7 (占7份) */
.login-left {
  flex: 7;
  min-width: 0;
}

/* 右侧：flex: 3 (占3份) */
.login-right {
  flex: 3;
  min-width: 0;
  justify-content: center;  /* 内容居中 */
}
```

**说明**：
- `flex: 7` + `flex: 3` = 总共10份
- 左侧占7/10 = 70%
- 右侧占3/10 = 30%

---

### 登录表单尺寸

```css
.login-box {
  width: 100%;
  max-width: 360px;      /* 最大宽度360px */
  padding: 40px 20px;    /* 上下40px，左右20px */
}
```

**说明**：
- 30%的右侧区域内
- 表单最大360px
- 两侧留有适当边距

## ✅ 优势说明

### 7:3比例的优势

**为什么选择7:3？**

1. **信息展示充分** ✅
   - 左侧70%空间足够展示品牌和特性
   - 2列网格布局舒适

2. **登录区域适中** ✅
   - 右侧30%对于登录表单来说刚好
   - 不会太宽（浪费），不会太窄（拥挤）

3. **视觉黄金比例** ✅
   - 7:3接近黄金分割比例（约0.618）
   - 视觉更加和谐

4. **焦点突出** ✅
   - 左侧作为主要展示区
   - 右侧登录区域清晰明确

---

### 居中 vs 靠左对比

| 对齐方式 | 优势 | 适用场景 |
|---------|------|---------|
| **靠左** | 视线自然流动 | 宽屏（50%+右侧） |
| **居中** | 视觉平衡 | 窄屏（30%右侧）✅ |

**选择居中的原因**：
- 右侧只有30%宽度
- 居中显示更加平衡
- 表单不会太靠边缘
